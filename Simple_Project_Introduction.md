# AI Tutor: Simple Project Introduction

## 🎓 **What is AI Tutor?**

**AI Tutor** is an intelligent learning platform that helps students learn better using artificial intelligence.

### **Key Features:**
- **Smart Chat** - Ask questions and get answers based on your study materials
- **Auto Quiz** - Generate practice questions from your documents automatically
- **Document Upload** - Upload PDF, Word, or text files for AI to learn from
- **Instant Help** - Get immediate answers to your questions

---

## 🚀 **How It Works**

### **Simple Process:**
1. **Upload** your study materials (books, notes, PDFs)
2. **Ask** questions about the content
3. **Get** intelligent answers based on your materials
4. **Generate** quizzes to test your knowledge
5. **Track** your learning progress

### **Technology:**
- **RAG System** - AI reads your documents and answers questions
- **Groq AI** - Fast, intelligent language processing
- **Streamlit** - Easy-to-use web interface
- **Python** - Reliable backend system

## 🛠️ **Technical Details**

### **Core Components:**
- **Frontend** - Streamlit web interface
- **Backend** - Python application
- **AI Engine** - Groq AI with RAG technology
- **Document Processor** - Handles PDF, DOCX, TXT files
- **Vector Database** - Stores and searches content

### **Dependencies:**
- Streamlit (web framework)
- Groq (AI API)
- PyPDF2 (PDF processing)
- Sentence Transformers (text understanding)
- NumPy (data processing)

## 🚀 **Getting Started**

### **Quick Setup:**
1. Install Python packages: `pip install -r requirements.txt`
2. Add your Groq API key to `.streamlit/secrets.toml`
3. Run the app: `python -m streamlit run app_simplified.py`
4. Open browser: `http://localhost:8501`

### **First Use:**
1. Upload your study documents
2. Click "Build RAG Index" to process files
3. Start asking questions in the chat
4. Generate quizzes from your content
